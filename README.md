# COSTCrypto

Собственные реализации некоторых криптографических ГОСТовых алгоритмов на Julia

## Текущее содержание

В настоящий момент реализованы следующие механизмы.

1. Алгоритмы блочного шифрования:
- Магма (с контрольными примерами)
- Кузнечик (с контрольными примерами)

2. Режимы шифрования:
- CBC (+ контрольные примеры для Магмы)
- CMAC (+ контрольные примеры для Магмы)

## Планы

In no particular order


### Симметричное

- Все стандартные режимы работы (CTR, CFB, OFB)
- Сопутствующие криптографические механизмы, в первую очередь различные KDF
- Более сложные режимы работы (MGM, CTR-ACPKM, OMAC-ACPKM)
- Шифрование диска: DEC, XEH, XTS и иные
- Режимы работы с конференций: sMGM, MGM2, MXP
- Хэширование
- Режимы работы хэша: HMAC, StreebogK

### Асимметричное 

- Группа точек эллиптической кривой
- VKO and friends
- Подпись 
- Усеченные варианты подписи
- Постквантовые подписи: коды
- Постквантовые подписи: решетки
- Постквантовые подписи: хэши
- Постквантовый KEM: коды
- Постквантовый KEM: решетки
